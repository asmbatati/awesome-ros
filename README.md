# Awesome ROS Ecosystem

A community-maintained database of ROS-related resources including:

- Packages
- Simulation tools
- Datasets
- Research papers
- Surveys
- Frameworks
- Tutorials

The goal is to create a structured dataset of the ROS ecosystem that can be used for research, discovery, and tooling.

---

## Dataset

The dataset is stored in CSV format under `/data`.

Example fields:

- name
- category
- description
- repository
- ros_version
- paper
- dataset
- simulation
- license

---

## Contributing

1. Fork the repository
2. Edit the CSV files in `/data`
3. Submit a Pull Request

Please follow the schema and avoid duplicates.
